# Python Basics

This repository contains basic Python programs for beginners.
It is created to practice core Python concepts and build a strong foundation.

## Topics Covered
- Variables and Data Types
- Conditional Statements (if-else)
- Loops (for, while)
- Functions
- Basic Programs

## Files Included
- hello.py
- star.py
  

## How to Run the Code
1. Install Python 3.x on your system
2. Open terminal / command prompt
3. Run any file using:
   python filename.py

Example:
python hello.py

## Purpose
This project is made for learning Python basics and for practice before internships and interviews.
