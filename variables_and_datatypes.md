# Variables and Data Types in Python

## Variable
A variable is used to store data in a program.
In Python, we do not need to declare the type of a variable.

Example:
x = 10
name = "Aayesha"

## Data Types
Data types define the type of data stored in a variable.

### 1. Integer (int)
Stores whole numbers.
Example:
x = 10

### 2. Float (float)
Stores decimal numbers.
Example:
y = 3.14

### 3. String (str)
Stores text or characters.
Example:
name = "Python"

### 4. Boolean (bool)
Stores True or False values.
Example:
is_active = True
